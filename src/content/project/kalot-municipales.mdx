# Kalot Municipales

[Kalot](https://kalot.fr) c'est une plateforme de compétition fun pour départager les musiques de campagne électorale en Guadeloupe.

Le nom vient de "kalotèr" en créole guadeloupéen, qui signifie querelle, chamaillerie, conflit. Parfait pour une compétition de musiques électorales !

## Le concept

Pendant les élections municipales, chaque candidat sort sa musique de campagne. C'est une tradition en Guadeloupe. Mais comment savoir laquelle est la meilleure ?

Kalot permet aux Guadeloupéens de voter pour leurs musiques préférées via un système de duels addictif.

## Comment ça marche

Le système de vote fonctionne par paires : deux chansons s'affrontent, l'utilisateur choisit sa préférée. Le gagnant reste en lice pour le prochain duel - une mécanique "winner-stays-on" qui rend le vote super engageant.

Un classement Elo (le même système utilisé aux échecs) est calculé en temps réel pour établir le leaderboard officiel. Les musiques les plus appréciées remontent naturellement dans le classement.

## Les features

- **Vote par duels** : Deux chansons, un choix, c'est simple et efficace
- **Winner-stays-on** : Le gagnant reste pour le prochain duel
- **Classement Elo** : Un leaderboard temps réel basé sur un algorithme reconnu
- **Leaderboard live** : Suivez l'évolution des musiques en direct
- **Modération** : Système de signalement pour garder le contenu approprié
- **Backoffice admin** : Interface d'administration pour gérer les chansons

## L'histoire du lancement

Lancé le 7 mars 2026 à 5h du matin avec 7 chansons au démarrage.

48h plus tard : **3.36k visiteurs**, **20+ chansons**, **~40 utilisateurs concurrents**.

La viralité locale a été immédiate - le concept a clairement touché une corde sensible auprès des Guadeloupéens en période électorale !

## Stack technique

Côté technique, c'est du moderne :
- **TanStack Start** avec React 19 pour le fullstack
- **Tailwind CSS 4** et **shadcn/ui** pour le design
- **Clerk** pour l'authentification sociale
- **Drizzle ORM** avec **Turso** pour la base de données
- **PostHog** pour les analytics
- Déployé sur **Netlify** avec Vite 7

Un projet fun qui a pris une vie propre en quelques jours ! 🎵🗳️
